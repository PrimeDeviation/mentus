Obsidian Local REST API Setup

  1. Install the "Local REST API" plugin in Obsidian.
  2. Enable the plugin and configure it with an API key.
  3. Copy the API key and endpoint URL to the fields above.
  4. In Obsidian, go to the Local REST API plugin settings and find the "HTTPS Certificate" section.
  5. Click on "Open Certificate" to view the certificate file.
  6. Install this certificate in your browser or operating system's trust store.
  7. Restart your browser after installing the certificate.

For detailed instructions on installing the certificate, please refer to your browser or operating system's documentation.

Note: The Obsidian Local REST API certificate is included in this extension. No manual certificate installation is required.
